A comprehensive repository dedicated to mastering Data Structures and Algorithms using Java. This project serves as a structured learning path, progressing from fundamental logic to complex problem-solving patterns.

---

## 📂 Repository Architecture

The repository is organized by data structure and algorithmic technique to ensure a modular learning experience.

### 🧱 1. Foundations & Logic (`Basics/`)
- **Core Syntax**: `Conditionals`, `Loops`, and `Methods` (Functions).
- **Pattern Printing**: Logical building through 2D coordinate-based patterns.
- **OOPS**: Object-Oriented Programming principles integrated with DSA.

### 📊 2. Linear Data Structures
- **Arrays**: `1D_Arrays`, `2D_Arrays`, and `Strings` manipulation.
- **Linked Lists**: Implementation of Singly, Doubly, and Circular lists.
- **Stacks & Queues**: Linear data management using LIFO and FIFO principles.

### 🔄 3. Algorithms & Optimization
- **Searching**: `Binary_Search` implementations ($O(\log n)$).
- **Sorting**: 
    - **Basic**: Bubble, Selection, and Insertion Sort.
    - **Advanced**: `Merge_Sort`, `Cyclic_Sort` (for range-based problems).
- **Recursion**: Solving complex problems through functional breakdown and backtracking.

### 🌲 4. Non-Linear Data Structures
- **Trees**: `Binary_Trees` fundamentals and traversals.
- **Search Trees**: `Binary_Search_Tree` (BST) operations and optimizations.
